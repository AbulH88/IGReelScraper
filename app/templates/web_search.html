Run a Web Search

Search for any keyword (e.g. "funny cats"). The app will extract all Reel URLs it can find on the web.

How it works

Web Search does not use the Instagram API. It searches Google/DuckDuckGo for your keyword and grabs any Instagram Reel links it finds.

  • Fast: No Instagram login needed. No rate limits.
  • Pending State: Initially, these reels will just be URLs (no views, no thumbnails).
  • Enrichment: You must click the Refresh button on the reels (or use 'Refresh All' in the Library) to command the app to visit the URL and pull the real-time views, likes, and video player.
